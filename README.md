## 🎯 🌟 About the Game

🎮 **TD_21 Game Python** is an interactive command-line game built using Python where you compete against the computer in a strategic counting challenge.

💀 The twist?
👉 Whoever lands on **21 loses**
👉 And must complete a **Truth 🤔 or Dare 😈**

---

## 🕹️ 🎲 Gameplay Rules

✨ Count starts from **1 → 21**
🤖 You vs Computer
🔢 Each turn, choose **1 to 3 numbers**
💥 Whoever says **21 loses**

🎭 Loser must choose:

* 🤔 Truth → Answer a question
* 😈 Dare → Complete a challenge
